🚀 Automated Executive Sales Dashboard (n8n + JavaScript)
📌 Overview

Dashboard Link : file:///C:/Users/imd90/Downloads/advanced_sales_dashboard%20(6).html

This project demonstrates an end-to-end workflow automation system that transforms raw sales data into an executive-level dashboard using n8n and JavaScript.

The workflow automates the complete pipeline:

Data cleaning
KPI calculation
Insight generation
Chart visualization
HTML report creation
Email delivery

The final output is a professional HTML dashboard designed for business decision-making.

⚙️ Features
Automated data cleaning and preprocessing
KPI calculation (Sales, Profit, Orders, Quantity, Discount, Profit Margin)
Monthly sales and profit trend analysis
Segment-wise performance insights
Region-wise sales vs profit analysis
Top-performing products and sub-categories
Shipping cost analysis
Discount vs profit relationship analysis
Dynamic chart generation
Automated HTML dashboard report generation
Email delivery automation
🧠 Workflow Architecture
Google Sheets / Input Data
        ↓
Data Cleaning (JavaScript)
        ↓
Data Processing & KPI Calculation
        ↓

        ↓
HTML Report Creation
        ↓

        ↓
Email Delivery (Gmail)
